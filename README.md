Guess the Number Game-
Python game where the player tries to guess a randomly generated number between 1 and 100. 
Includes hints for close guesses and tracks the number of attempts.
How to Run-
Make sure you have Python installed, then run:
```bash
python guessanum.py
Features-
- Random number generation
- Input validation
- Hint system for near guesses
- Attempt counter
-Friendly messages and emoji feedback
